"build.base.bat": Compress and Exporting all your released code in to "Build" folder. 
"build.base.Encrypted.bat": Encrypt the code in the "Build" folder. You need to execute "build_base.bat" first.

"build.Standalone.bat": Copying all the code from "Build" to "Standalone". You need to "build" your code first.
PS-> "Standalone" folder is the folder for CI to deploy all the code on our web server.
